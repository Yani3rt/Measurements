# Ghost Accent Trace Motion Design

## Goal

Refine selected measurement animation so the path draw is legible and expressive before the selected state fully settles in.

## Approved Direction

Use a ghost-to-accent trace:

- keep the inactive path visible underneath
- start the thin bright petrol trace immediately on selection
- let the full selected stroke settle in a beat later
- finish with the measurement line itself in teal
- let the anchor pulse and chip emphasis wake up after the trace begins

## Sequence

1. base inactive measurement path remains visible
2. thin guidance trace draws along the path immediately
3. selected accent stroke settles in behind it
4. anchor pulse and chip lift reinforce the completed selection

## Success Criteria

- users can visually follow the selected measurement path
- the draw reads clearly before the selected line becomes heavy
- the result feels refined and readable rather than abrupt
